## Meetmax

## Social Network Application with Next.js 14 App Router: React, Tailwind, NextAuth

Features:
- Tailwind design
- Full responsiveness
- Credential authentication
- SignUp
- SignIn

### Cloning the repository

```shell
git clone https://github.com/masrur-sakib/meetmax.git
```

### Install packages

```shell
npm i or npm install
```

### Start the app

```shell
npm run dev
```
